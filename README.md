# ItemWorth

ItemWorth is an offline-first Progressive Web App (PWA) for tracking personal inventory and estimated item values.

## Features

- Offline support
- Installable as a PWA
- Create, view, edit, and delete inventory items
- Local device storage
- Mobile-friendly interface
- Color-blind-friendly design
- Privacy Policy, Terms of Use, and About pages

## Tech Stack

- React
- Vite
- vite-plugin-pwa
- Workbox

## Running Locally

```bash
npm install
npm run dev
npm run build
npm run preview
```
